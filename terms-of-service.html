2. Description of Service

Mini Note is a Chrome browser extension designed to provide users with a quick, lightweight, and efficient note-taking solution. The Service allows users to capture thoughts, create notes, organize tasks, and sketch ideas directly within their web browser without the need for external applications or cloud accounts.

Key Features Include:

  • Quick Note-Taking: Instantly capture text notes with a single click or keyboard shortcut
  • Smart Search: Intelligent search functionality with fuzzy matching and filters
  • Task Management: Organize tasks using an intuitive drag-and-drop Kanban board
  • Drawing Canvas: Sketch ideas with pen, highlighter, and eraser tools
  • Local Storage: All data is stored locally on your device for privacy
  • Export Options: Export notes in various formats including text, markdown, and PDF
  • Themes: Customize the appearance with multiple color themes
  • Offline Access: Full functionality without internet connection

The Service is provided "as is" and is intended for personal productivity purposes. We continuously work to improve and expand the features available to users.
