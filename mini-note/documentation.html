Getting Started

Welcome to Mini Note! This guide will help you get up and running in just a few minutes.

Installation

  1. Visit the Chrome Web Store
  2. Search for "Mini Note" or click our install link
  3. Click "Add to Chrome"
  4. Confirm the installation when prompted

First Launch

After installation, you'll see the Mini Note icon in your browser toolbar. Click it to open your first note!

üí° Pro Tip: Pin Mini Note to your toolbar for quick access. Right-click the extension icon and select "Pin".

Creating Notes

Creating notes in Mini Note is quick and intuitive. Here's everything you need to know.

Quick Note

Click the Mini Note icon or use the keyboard shortcut to instantly start typing. Your note is saved automatically as you type.

New Note

To create a new note while keeping your existing ones:

  1. Click the "+" button in the toolbar
  2. Or use the keyboard shortcut Ctrl/Cmd + N

Formatting

Mini Note supports basic text formatting:

  • Bold - Select text and press Ctrl/Cmd + B
  • Italic - Select text and press Ctrl/Cmd + I
  • Lists - Start a line with "- " or "1. "

Smart Search

Find any note instantly with our intelligent search system.

Basic Search

Simply start typing in the search bar to find notes containing your query. Results appear instantly as you type.

Advanced Search Features

  • Fuzzy Matching - Find notes even with typos or partial words
  • Tag Search - Use #tagname to filter by tags
  • Date Filters - Search by creation or modification date
  • Content Type - Filter by notes, tasks, or drawings
Search examples: - "meeting" ‚Üí finds all notes containing "meeting" - "#work" ‚Üí finds notes tagged with "work" - "date:today" ‚Üí shows notes created today - "type:task" ‚Üí shows only task boards
üí° Pro Tip: Press Ctrl/Cmd + F from anywhere to instantly open search!

Task Management

Organize your tasks visually with our intuitive Kanban board.

Creating a Task Board

  1. Click the "+" button and select "New Task Board"
  2. Give your board a name
  3. Start adding tasks to your columns

Default Columns

Each new board comes with three columns:

  • To Do - Tasks waiting to be started
  • In Progress - Tasks you're currently working on
  • Done - Completed tasks

Drag and Drop

Simply drag any task card and drop it into another column to update its status. You can also reorder tasks within a column.

Task Features

  • Add due dates and reminders
  • Assign priority levels (Low, Medium, High)
  • Add checklists within tasks
  • Color-code tasks for quick identification
üí° Pro Tip: Double-click any column header to rename it, or add custom columns for your workflow!

Drawing Canvas

Sketch ideas, create diagrams, or annotate with our powerful drawing tools.

Opening the Canvas

  1. Click the "+" button and select "New Drawing"
  2. Or press Ctrl/Cmd + D for quick access

Drawing Tools

  • Pen - Freehand drawing with adjustable thickness
  • Highlighter - Semi-transparent strokes for emphasis
  • Eraser - Remove unwanted strokes
  • Shapes - Draw rectangles, circles, and lines
  • Text - Add text annotations to your drawings

Color Palette

Choose from a variety of colors for your pen and highlighter. Custom colors are also supported via the color picker.

Canvas Actions

  • Undo/Redo - Ctrl/Cmd + Z / Ctrl/Cmd + Shift + Z
  • Clear Canvas - Start fresh with one click
  • Export as Image - Save your drawing as PNG or JPG
  • Insert into Note - Embed drawings in your text notes
üí° Pro Tip: Use two-finger pinch to zoom in/out on the canvas for detailed work!

Organizing Notes

Keep your notes tidy with folders and tags.

Folders

Create folders to group related notes together. Click the folder icon in the sidebar to create a new folder.

Search

Use the search bar to quickly find any note. Search works across titles and content.

Search tips: - "meeting" - finds all notes containing "meeting" - "#work" - finds notes tagged with "work" - "folder:personal" - searches within a specific folder

Keyboard Shortcuts

Speed up your workflow with these handy shortcuts.

Ctrl/Cmd + Shift + N Open Mini Note
Ctrl/Cmd + N New Note
Ctrl/Cmd + S Save Note
Ctrl/Cmd + F Search Notes
Ctrl/Cmd + B Bold Text
Ctrl/Cmd + I Italic Text
Ctrl/Cmd + E Export Note
Escape Close Mini Note

Themes & Settings

Customize Mini Note to match your style.

Available Themes

  • Light - Clean white background, perfect for daytime use
  • Dark - Easy on the eyes for night owls
  • Sepia - Warm paper-like tones for comfortable reading
  • Ocean - Cool blue tones for a calming experience

Changing Themes

Click the settings gear icon and select your preferred theme from the dropdown menu.

Export & Backup

Never lose your important notes with our export features.

Export Options

  • Plain Text (.txt) - Universal format, works everywhere
  • Markdown (.md) - Keep your formatting intact
  • PDF - Perfect for sharing and printing
  • JSON - Full backup with all metadata

Auto Backup

Enable Chrome Sync in settings to automatically backup your notes across all your Chrome browsers.

Troubleshooting

Having issues? Here are solutions to common problems.

Notes not saving

Try clearing your browser cache and restarting Chrome. If the issue persists, reinstall the extension.

Extension not opening

Make sure Mini Note is enabled in Chrome's extension settings (chrome://extensions).
